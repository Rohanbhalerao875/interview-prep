## Application template

Subject line options:
- "Java Backend Engineer | 4 years Spring/Microservices | Ready for Remote/International"
- "[Your Name] — Senior Java Developer | AWS | System Design"

Body:
```
Hi [Hiring Manager / Recruiter],

I am [Name], a Java Backend Engineer with 4 years of experience building scalable 
microservices at [Current Company]. I have expertise in:
- Spring Boot and Spring Cloud
- AWS and distributed systems
- REST APIs and event-driven architecture
- High-performance systems (X TPS)

I am actively seeking remote or international roles in the $30-40K USD range. 
I am particularly interested in [specific tech stack or domain relevant to their job].

Let me know if this aligns with your current opening.

[LinkedIn URL]
[GitHub portfolio URL]

Best regards,
[Your Name]
```

## Salary negotiation notes

- Target: **$30-40K USD annually** (~25-35 LPA equivalent)
- Ranges to suggest: "30-45K USD" to leave negotiation room
- Factors:
  - 4 years experience → Senior Engineer / Backend II tier
  - Spring/microservices expertise
  - System design capability
  - Remote or relocation flexibility
